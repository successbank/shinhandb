## Hook 목록

```yaml
request_intake:
  trigger: 요청 접수 시
  action: 유사 요청 탐지, 중복 방지
  developed_by: 최예린

pre_commit:
  trigger: git commit 전
  action:
    - ESLint/Prettier 검증
    - TypeScript 타입 체크
    - 테스트 실행
    - 네이밍 규칙 검증
  developed_by: 윤재호

code_review:
  trigger: PR 생성 시
  action:
    - 자동 체크리스트 생성
    - 변경 범위 분석
    - 리뷰어 자동 배정
  developed_by: 윤재호

post_deploy:
  trigger: 배포 완료 후
  action:
    - 헬스체크
    - 스모크 테스트
    - 모니터링 연동
  developed_by: 박성민
```
